# Анализ A/B теста: Z-тест равенства выборок

## 📚 Обзор

Был проведен A/B тест, целью которого было оценить влияние нового функционала на определённую метрику. Тест включал две группы: **контрольную** и **тестовую**, по 5000 участников в каждой. Основной задачей было определить, оказало ли нововведение статистически значимое влияние на выбранную метрику.

---

## 🔍 Легенда

- **Оцениваемая метрика**: Одна ключевая метрика.  
- **Группы**:
  - **Контрольная группа**: 5000 участников.  
  - **Тестовая группа**: 5000 участников.

---

## 📊 Гипотезы

- **Нулевая гипотеза (H₀)**: Нововведение **не влияет** на метрику (\( \mu_{control} = \mu_{test} \)).  
- **Альтернативная гипотеза (H₁)**: Нововведение **влияет** на метрику (\( \mu_{control} \neq \mu_{test} \)).  

---

## 🧮 Статистический тест

Был проведён **Z-тест равенства выборок** для сравнения средних значений контрольной и тестовой групп.

- **Наблюдаемое значение Z (Z_набл)**: `1.151040706`  
- **Критическое значение Z (Z_табл)**: `±1.96` (двусторонний тест, \( \alpha = 0.05 \))  

---

## 📈 Вывод

- При уровне значимости (\( \alpha = 0.05 \)) наблюдаемое значение Z **не превышает** критическое (\( Z_{набл} < Z_{табл} \)).  
- **Результат**: Мы принимаем нулевую гипотезу (H₀).  
- **Интерпретация**: Нет статистически значимых доказательств того, что нововведение повлияло на метрику.

---

## 🚀 Основные выводы

Этот анализ демонстрирует процесс проверки гипотез в рамках A/B тестов и подчёркивает важность статистической строгости при интерпретации результатов. Несмотря на предположения, результаты показывают, что нововведение **не привело** к значимым изменениям метрики при заданном уровне доверия.
